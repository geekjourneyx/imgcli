# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2026-03-10

### Added

- Initial stable release of the Agent-native image post-processing CLI.
- Core commands: `inspect`, `compose`, `convert`, `variants`, `run`, `smartpad`, `topdf`, and `stitch`.
- Structured JSON success and error responses with stable error codes.
- CI, release-check, install script, e2e coverage, and real smoke validation.
